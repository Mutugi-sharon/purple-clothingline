# Purple Style Clothing Website

## Overview
Purple Style Clothing is a simple, modern clothing website built using **HTML** and **CSS**.  
It showcases categories, featured products, fashion videos, and includes contact information.  
The color theme is **purple and white** for a stylish and elegant look.

---

## Features

- **Home Page** with a welcome section
- **Categories** section for Women, Men, and Accessories
- **Featured Products** section with images, names, and prices
- **Fashion Videos** section with embedded YouTube videos
- **Delivery & Pick Up** information
- **Contact Information** with phone and email
